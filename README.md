# vscode-sciter 
Add [[Sciter]](https://sciter.com) support for vscode.  
tmLanguage files source: [https://github.com/svitalij/sciter.tm.syntax](https://github.com/svitalij/sciter.tm.syntax)

# Features
Colorizers.

``` json
{
    "files.associations": {
		"*.css": "sciter-css",
		"*.html": "sciter-html",
		"*.htm": "sciter-html"
	}
}
```
